# ACRE Manifesto

> "形なきものに、仮の宿を与える。"

## 01. Context (背景)
なぜ今、ACREが必要なのか。
大量生産・大量消費のサイクルの中で摩耗していく「モノ」への違和感。
ソフビ（Soft Vinyl）という素材が持つ「中空（Empty）」な性質と、現代の虚無感のリンクについて。

## 02. Core Principles (基本原則)
1. **Anti-Mass Production**
   我々は効率化された工場ではなく、意味を醸成する実験室を持つ。
2. **Figuration as Ritual**
   造形とは、単なる出力作業ではなく、見えない領域に触れる儀式である。
3. **Origins**
   全ての創造はゼロ地点（ACRE）から始まる。
