# ACRE（アケル）

ACREは、**「造形物（Figure）」を"形"ではなく、非存在（Urun/Orah）への接続（Figuration）の結果として扱う**ためのフレームです。  
ここではACREの**思想（マニフェスト）**と**定義（用語・モデル）**を公開し、改訂履歴込みで管理します。

## このリポジトリの目的
- ACREの定義を曖昧にしない（用語を固定する）
- 議論と改訂を"履歴"として残す
- 作品（Figure）がACREを検証できる状態をつくる
