# ACRE: Art Theory Framework

A comprehensive framework for art theory, criticism, and analysis.

## Overview

ACRE is a documentation-based framework that provides structured approaches to understanding, analyzing, and discussing art across various mediums and historical contexts.

## Project Structure

```
ACRE/
├── docs/               # Documentation files
│   ├── introduction/   # Introduction to ACRE framework
│   ├── theory/         # Core theoretical concepts
│   ├── methods/        # Analytical methods and approaches
│   ├── case-studies/   # Applied examples and case studies
│   └── resources/      # Additional resources and references
├── README.md           # This file
└── mkdocs.yml         # MkDocs configuration
```

## Getting Started

### Prerequisites

- Python 3.7+
- pip

### Installation

1. Clone the repository:
```bash
git clone https://github.com/oltecshota-lgtm/ACRE.git
cd ACRE
```

2. Install MkDocs and dependencies:
```bash
pip install mkdocs mkdocs-material
```

3. Serve the documentation locally:
```bash
mkdocs serve
```

4. Build the static site:
```bash
mkdocs build
```

## Documentation

Visit the documentation at `http://localhost:8000` when running `mkdocs serve`.
