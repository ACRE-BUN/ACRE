# 用語集（読み方＋定義）

## ACRE構造
- **ACRE（アケル）**：非存在への接続のされ方を扱うフレーム
- **Nul（ヌル）**：分岐以前の未分化
- **第一分岐（だいいちぶんき）**：存在／非存在の分化
- **第二分岐（だいにぶんき）**：非存在がOrah/Urunへ分化

## 非存在
- **非存在（ひそんざい）**：存在として確定しない領域
- **Orah（オーラ）**：外側の非存在（アクセス不可能領域）
- **Urun（ウルン）**：内側の非存在（主体に沈む領域）

## 接続と外在化
- **接続（せつぞく）**：理解ではなく、主体構造が再編成される出来事
- **圧力（あつりょく）**：Urunが意味化以前に作用する形
- **Figuration（フィギュレーション）**：接続を外在化する最小操作
- **Figure（フィギュア）**：外在化の成果物（造形物の総称）

## 分類
- **Proto-Figuration（プロト・フィギュレーション）**：不可避の外在化
- **Reflexive Figuration（リフレクシブ・フィギュレーション）**：再接続としての外在化
