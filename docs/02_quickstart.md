# クイックスタート（3分）

## ACREの最短セット
- **Urun**：内側の非存在
- **接続**：主体がUrunにより再編成される出来事
- **Figuration**：接続を外在化する最小操作
- **Figure**：外在化の成果物（造形物）

## 接続の三段階（超要約）
1. **兆候**：意味になる前の圧力が立つ
2. **閾値**：回避不能になり、再編成が起きる
3. **外在化**：最小操作として形が生まれる

## Proto / Reflexive の違い（超要約）
- **Proto**：不可避（作らない自由が消える）
- **Reflexive**：再接続（作らない自由がある中で選ばれる）
